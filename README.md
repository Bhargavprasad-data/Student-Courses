# Student Courses Web Application

This is a web application for managing student courses and registrations. The application provides a user-friendly interface for students to sign up, log in, and manage their course enrollments.

## Project Structure

```
├── funct.js         # JavaScript functionality and business logic
├── Home.html        # Home page of the application
├── Login.html       # User login page
├── Signup.html      # New user registration page
└── style.css        # CSS styles for the application
```

## Features

- User Registration: New students can create an account
- User Authentication: Secure login system for registered users
- Course Management: View and manage course enrollments
- Responsive Design: Mobile-friendly user interface

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Modern Web Standards

## Getting Started

1. Clone this repository
2. Open `Home.html` in your web browser to start using the application
3. Use the navigation to access the Login or Signup pages as needed
